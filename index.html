História da Web

(Voltar)

Os primeiros computadores surgiram na década de 1940 com o Mark1, Colossus e ENIAC. Eram máquinas muito grandes na qual realizavam cálculos. Inclusive, Colossus chegou a ser utilizado para realizar cálculos durante a Segunda Guerra. A medida que os anos passavam, os computadores evoluiam e diminuiam de tamanho até chegar aos computadores como são hoje.

Nos Estados Unidos na década de 1960, a internet surge com o objetivo de criar uma rede de comunicação de computadores em pontos estratégicos no qual as informações não pudessem ser destruídas por bombardeios. Basicamente, a internet era utilizada para fins de pesquisas e militares. Com o passar dos anos, a internet se popularizou e hoje faz parte da vida de praticamente toda população. A história da computação está dividida em quatro períodos:

  • PRIMEIRA GERAÇÃO

    • Os primeiros computadores funcionavam por meio de válvulas eletrônicas. As válvulas aqueciam bastante e acabavam queimando e dando problemas com facilidade. Os computadores dessa época eram muito grandes e consumiam bastante energia. Exemplo desses computadores: Mark1, Colossus e ENIAC.

  • SEGUNDA GERAÇÃO

    • Na segunda geração as válvulas foram trocadas por transitores, eles eram menores, consumiam menos energia, enquentavam menos e eram mais rápidos e confiáveis. Nesse período, o tamanho dos computadores já haviam diminuído consideravelmente.

  • TERCEIRA GERAÇÃO

    • A terceira geração é marcada pelo uso dos microchips, o que possibilitou a construção de equipamentos menores e mais baratos. Foi nesse período que a utilização de computadores pessoais começou.

  • QUARTA GERAÇÃO

    • Esse período é marcado pelo surgimento dos processadores com gasto de energia cada vez menor. Sistemas opercionais como MS-DOS, UNIX, Apple’s Macintosh foram construídos e linguagens de programação como C++ e Smalltalk foram desenvolvidas. Os computadores eram confiáveis, mais rapidos e possuiam maior capacidade de armazenamento. Nesse período, a partir da decáda de 90, há um aumento na venda de computadores pessoais. Atualmente estamos na quarta geração.

    O que são Clients e Servers

    (Voltar)

    O client é o dispositivo que está consumindo a informação na internet. Na computação, pode ser representada como uma entidade que consome os serviços de uma entidade servidora, através do uso de uma rede de computadores. Através do client se usa navegadores para acessar um servidor.

    Os servidores são computadores capazes de processar aplicações, prestar serviços e armazenar dados. Possuem alto poder de processamento e sao capazes de excutar conjuntos específicos de programas para fornecer serviços para outras máquinas ou clientes.

    O que é Linguagem de Programação

    (Voltar)

    É uma linguagem formal que recebe e executa uma série de instruções, símbolos, palavras-chaves,regras semânticas e sintáticas. Permite que um programador crie programas a partir de um conjunto de ordens, ações consecutivas, dados e algoritmos. Javascript, Python, Java, PHP, C#, são exemplos de linguagens usadas na programação. Algumas linguagens são server-side que são usadas nos servidores, e outras linguagens são client-side que sao utilizadas no cliente.

    Introdução ao HTML

    (Voltar)

    HTML significa HiperText Markup Language e é o componente básico da web, ele estabelece a estrutura básica de um website. Sem ele o navegador não saberia exibir imagens, textos e outros conteúdos. O HTML não é uma linguagem de programação, mas sim uma linguagem de marcação. O código pode ser escrito através de qualquer editor de texto, como o próprio bloco de notas. Cada página consiste em uma série de tags que podem ser considerados os blocos de construção das páginas. Portanto, esses blocos são a maneira com a qual o HTML faz a marcação dos conteúdos, criando a hierarquia e a estrutura do mesmo, dividido entre seções, parágrafos, cabeçalhos, e outros.

    As tags são códigos que constroem todas as estrutura de uma página, dando forma ao site. Existem vários tipos de tags e algumas das mais utilizadas são:

    • "head"- São as informações que serão carregadas antes de ser exibido ao usuário;
    • "body"- É onde ficará tudo que será exibido para o usuário;
    • "title"- Adiciona o título;
    • "strong"- Deixa o texto em negrito;
    • "i"- Deixa o texto em itálico;
    • "u"- Sublinha o texto;
    • "h1,h2,h3,h4,h5,h6"- Define o título e os subtítulos;
    • "ol"- Define uma lista ordenada;
    • "ul"- Define uma lista não ordenada;
    • "img"-Adiciona imagens;
    • "blockquote"- Adiciona uma citação;
    • "mark"- Marca o texto;
    • "p"- Define o paragráfo;
    • "input"- Define os campos do formulário;
    • "a"- Adiciona um link.
